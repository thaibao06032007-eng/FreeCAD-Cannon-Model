# 3D Printed Mini Cannon 💣

![Cannon Image](<img width="1309" height="990" alt="Screenshot 2026-03-30 231820" src="https://github.com/user-attachments/assets/140731cc-ea89-4277-99dd-5d03a0630006" />)

A modular desktop mini-cannon model designed from scratch using **FreeCAD 1.1**. This project was created to practice 3D modeling skills, explore the features of the new FreeCAD update, and learn about tolerances in 3D printing.

## 🗂️ Files Included
All parts have been exported as high-quality `.3mf` files. The original FreeCAD workspace file (`Canon.FCStd`) is also included if you want to modify or remix the design!

To build one complete cannon, you will need to print the following quantities:
* **1x** Cannon Barrel
* **2x** Side Plates
* **4x** Wheels
* **2x** Wheel Axles
* **1x** Middle Axle
* **1x** Trunnion

## 🖨️ Recommended Print Settings
To ensure the axles fit perfectly into the wheels and side plates, please use the following settings:
* **Layer Height:** `0.2 mm` (Crucial for smooth curves and accurate assembly tolerances).
* **Infill:** `15% - 20%` (Gyroid or Grid).
* **Material:** PLA, PETG, or ABS.
* **Supports:** Minimal to none required if parts are oriented flat on the build plate.

## 🛠️ Assembly
All parts are designed to be friction-fit. Simply slide the axles through the side plates and attach the wheels. The trunnion holds the cannon barrel in place between the two plates.
